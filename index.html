How to set up Metamask for trading
Smudgecoin on CarbonSwap

To trade Smudgecoin on Carbonswap.exchange you need to configure your Metamask so it runs on EnergyWebChain (EWC).
If you need more help please read more at docs.carbonswap.exchange

Get some EWT (EnergyWeb Tokens) if you don’t already have it. You can buy it at centralized exchanges like Kraken and Kucoin and send them to your wallet (be sure to send them on the EnergyWeb Chain, not on Ethereum - this is a common mistake among newcomers)

To trade goto carbonswap.exchange/#/swap
and swap your EWT’s to an optional amount of Smudge tokens.

That’s it! Now you know how to trade Smudgecoin on Carbonswap!

To trade on Carbonswap you need to download and use Metamask. Download it at https://metamask.io/
The next step you need to configure Metamaks so it runs on EnergyWebChain (EWC).

Configure your Metamask wallet with these parameters:
Network name: EWC
New RPC URL: https://rpc.energyweb.org
Chain ID: 246
Currency Symbol: EWT
Block Explorer URL: https://explorer.energyweb.org
